Cognifyz Full Stack Development Internship Project

Project Title
Student Management System

Description
A Full Stack Student Management System developed during the Cognifyz Full Stack Development Internship. The application allows users to register, log in, manage student records, and perform CRUD operations using Node.js, Express.js, MongoDB, and EJS.

Features
- User Registration and Login
- Session-based Authentication
- Student Registration Form
- Form Validation
- Create, Read, Update, Delete (CRUD) Operations
- MongoDB Database Integration
- Responsive User Interface
- Dynamic DOM Manipulation
- REST API Implementation

Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Node.js
- Express.js
- EJS
- MongoDB
- Mongoose
- Express Session
- bcryptjs
- Git & GitHub

Installation

Clone Repository

```bash
git clone https://github.com/madhurathod2005-stack/cognifyz-fullstack-.git
```

Install Dependencies

```bash
npm install
```

Run Project

```bash
node server.js
```

or

```bash
nodemon server.js
```

Open Browser

```text
http://localhost:3000
```
